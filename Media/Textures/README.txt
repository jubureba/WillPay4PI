Will Pay 4 PI - Custom Textures
================================

Place custom texture files in this folder.
Supported formats: .tga, .blp
Recommended sizes: 32x32, 64x64, 128x128, 256x256

Reference them as:
  Interface\AddOns\PIAssistant\Media\Textures\your_file
  (without file extension for .tga/.blp)
